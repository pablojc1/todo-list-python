# Gerenciador de Tarefas

Aplicação de gerenciamento de tarefas desenvolvida em Python puro e HTML/CSS/JS.

## Como testar

### Versão terminal (Python)
1. Instale o Python em python.org
2. Baixe o arquivo `todo.py`
3. Abra o terminal na pasta do arquivo e rode:

4. ### Versão web (Frontend)
1. Baixe o arquivo `todo.html`
2. Abra ele direto no navegador (dois cliques)
3. Pronto — sem instalar nada

## Funcionalidades
- Adicionar tarefas
- Listar tarefas com status
- Marcar como concluída
- Deletar tarefa

## Tecnologias
- Python 3
- HTML, CSS, JavaScript
- LocalStorage (persistência de dados no frontend)
